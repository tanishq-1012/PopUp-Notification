# PopUp Notification

A simple Popup Notification component built using HTML and CSS. This project demonstrates how to create a visually appealing notification pop-up that can be used on websites to notify users of new updates, purchases, messages, or any other important information.

# Features

* Responsive Design**: The notification popup is fully responsive and works well on both desktop and mobile devices.
* Smooth Animation**: A CSS animation is used to smoothly animate the popup, enhancing the user experience.
* Customizable Styles**: Easily modify colors, fonts, sizes, and other styles to fit your website's design.

# Technologies used

* HTML**: Provides the structure of the popup notification.
* CSS**: Used for styling and animating the popup.
